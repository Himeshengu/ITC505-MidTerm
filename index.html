Addendum

  • HTML: The story is structured using basic HTML elements such as <div>, <p>, and <button>. These elements help organize the content into sections such as the story text, choices, and the restart button.
  • CSS: Styling is applied through an external CSS file that defines the layout, colors, and positioning of elements. Flexbox is used to center the content, and background images are applied for visual appeal. The footer is positioned to the right side of the page using position: absolute;.
  • JavaScript: The script.js file handles the logic of the interactive story. It listens for user inputs (button clicks) to change the story text and image dynamically. Additionally, the last modified date of the page is shown using JavaScript.
